This case study is all about data interrogation with the aim to find out which products generates the most revenue, the time of day that revenue is maximized and what the trends are across the products and time intervals as well as the recommendation to improve the sales

I used the data that was provided, planned using MS Excel for Gantt chart and Miro for flowcharts

Databricks was used for data processing SQL scrips Miro was used for planning using flow charts MS Excel was used for Gantt charts since Canva needs subscription MS Powerpoint was used for the final presentation
